Component 1: NGFW

Don't build a firewall engine.

Use:

Linux nftables
eBPF/XDP

XCloak-Security-Suite NGFW should become:

XCloak-Security-Suite UI
    ↓
Policy Engine
    ↓
nftables

Store rules in PostgreSQL.

Example:

{
  "src":"10.0.0.0/24",
  "dst":"any",
  "port":"443",
  "action":"allow"
}

Generate nftables rules automatically.
